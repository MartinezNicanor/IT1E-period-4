### :running: Run Locally

If you want to run the web application locally please follow the information below.

Clone the project

```bash
  git clone https://github.com/MartinezNicanor/IT1E-period-4.git
```

Go to the project directory

```bash
  cd IT1E-period-4
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
